## RU
**Chat Cleaner** - блокирует радио, награды и уведомления, поддерживает кастомные сообщения (вход/выход, смена команды). Настройка через конфиг.

**Требования:**
- [Utils](https://github.com/Pisex/cs2-menus/releases)

## EN
**Chat Cleaner** - blocks radio commands, rewards, and notifications. Supports custom messages (join/leave, team switch). Configurable via config files.

**Requirements:**
- [Utils](https://github.com/Pisex/cs2-menus/releases)
